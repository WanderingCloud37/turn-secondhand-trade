# 第1章 项目概述

## 1.1 我们要做什么

做一个 Windows 桌面程序，让大学生可以在上面**买卖二手物品**。

**核心功能链路（必须全部实现）：**

```
注册账号 → 登录系统 → 浏览商品列表 → 发布自己的商品
                                              ↓
                       查看商品详情 → 下单购买 → 在我的订单中确认收货
```

## 1.2 技术栈

| 技术 | 说明 |
|------|------|
| C# .NET 10 | 编程语言和运行环境 |
| Windows Forms | 桌面界面框架（拖控件做界面） |
| Microsoft.Data.SqlClient | 连接 SQL Server 数据库 |
| SQL Server | 存数据的数据库 |
| Visual Studio 2022+ | 写代码的工具 |

# 第3章 数据库设计

## 3.1 E-R 关系

| 表 A | 关系 | 表 B | 说明 |
|------|------|------|------|
| `users`（用户） | 1 → N | `goods`（商品） | 一个用户可以发布多个商品 |
| `users`（用户） | 1 → N | `orders`（订单-买家） | 一个用户可以下多个订单 |
| `users`（用户） | 1 → N | `orders`（订单-卖家） | 一个用户可以收到多个订单 |
| `categories`（分类） | 1 → N | `goods`（商品） | 一个分类下有多个商品 |
| `goods`（商品） | 1 → N | `orders`（订单） | 一个商品可以有多个订单记录 |

## 3.3 完整 SQL 建库建表脚本

在 SQL Server Management Studio（SSMS）中**新建查询**，粘贴以下内容，按 F5 执行：

```sql
-- ==================== 创建数据库 ====================
IF NOT EXISTS (SELECT * FROM sys.databases WHERE name = 'SecondHandDB')
    CREATE DATABASE SecondHandDB;
GO

USE SecondHandDB;
GO

-- ==================== 1. 用户表 ====================
CREATE TABLE users (
    user_id     INT           IDENTITY(1,1) PRIMARY KEY,
    username    NVARCHAR(50)  NOT NULL UNIQUE,
    password    NVARCHAR(50)  NOT NULL,
    phone       NVARCHAR(20)  NULL
);
GO

-- ==================== 2. 分类表 ====================
CREATE TABLE categories (
    category_id   INT          IDENTITY(1,1) PRIMARY KEY,
    category_name NVARCHAR(50) NOT NULL UNIQUE
);
GO

-- ==================== 3. 商品表 ====================
CREATE TABLE goods (
    goods_id     INT            IDENTITY(1,1) PRIMARY KEY,
    title        NVARCHAR(100)  NOT NULL,
    price        DECIMAL(10,2)  NOT NULL,
    description  NVARCHAR(MAX)  NULL,
    image_url    NVARCHAR(200)  NULL,
    category_id  INT            NOT NULL REFERENCES categories(category_id),
    seller_id    INT            NOT NULL REFERENCES users(user_id),
    status       NVARCHAR(10)   NOT NULL DEFAULT '在售',
    created_time DATETIME       NOT NULL DEFAULT GETDATE()
);
GO

-- ==================== 4. 订单表 ====================
CREATE TABLE orders (
    order_id      INT          IDENTITY(1,1) PRIMARY KEY,
    goods_id      INT          NOT NULL REFERENCES goods(goods_id),
    buyer_id      INT          NOT NULL REFERENCES users(user_id),
    seller_id     INT          NOT NULL REFERENCES users(user_id),
    order_status  NVARCHAR(10) NOT NULL DEFAULT '待收货',
    order_time    DATETIME     NOT NULL DEFAULT GETDATE(),
    complete_time DATETIME     NULL
);
GO

PRINT '数据库创建完成！';
```

**连接字符串各参数含义：**

# 第4章 系统模块与窗体设计

## 4.0 全局设计：Program.cs 全局变量

由于我们不用三层架构，登录后的用户信息需要一种方式在**所有窗体之间共享**。最简单的方式是在 Program.cs 里加两个全局变量。

**操作步骤：**

```
打开 Program.cs → 在 class Program 里面加上两行高亮的代码
```

```csharp
namespace 转一转校园二手物品交易系统
{
    internal static class Program
    {
        // ====== 新增：全局变量，存当前登录用户的信息 ======
        public static int CurrentUserId = 0;
        public static string CurrentUserName = "";
        // ============================================

        [STAThread]
        static void Main()
        {
            ApplicationConfiguration.Initialize();
            Application.Run(new FrmLogin());
        }
    }
}
```

**原理：** `public static` 的意思是"整个程序任何地方都能访问"。登录成功后给这两个变量赋值，其他窗体就能通过 `Program.CurrentUserId` 和 `Program.CurrentUserName` 读取当前是谁在操作。

```csharp
using Microsoft.Data.SqlClient;
using System.Data;

namespace 转一转校园二手物品交易系统
{
    public partial class FrmLogin : Form
    {
        public FrmLogin()
        {
            InitializeComponent();
        }

        private void btn_Login_Click(object sender, EventArgs e)
        {
            // 1. 检查是否为空
            if (string.IsNullOrEmpty(txt_UserName.Text) || string.IsNullOrEmpty(txt_Pwd.Text))
            {
                MessageBox.Show("请输入用户名和密码");
                return;
            }

            // 2. 查数据库：用户名和密码是否匹配
            string sql = "SELECT user_id, username FROM users WHERE username=@u AND password=@p";
            SqlParameter[] ps = {
                new SqlParameter("@u", txt_UserName.Text.Trim()),
                new SqlParameter("@p", txt_Pwd.Text)
            };
            DataTable dt = SQLHelper.Query(sql, ps);

            // 3. 没查到 → 用户名或密码错误
            if (dt.Rows.Count == 0)
            {
                MessageBox.Show("用户名或密码错误");
                return;
            }

            // 4. 查到了 → 保存用户信息到全局变量
            Program.CurrentUserId = Convert.ToInt32(dt.Rows[0]["user_id"]);
            Program.CurrentUserName = dt.Rows[0]["username"].ToString();

            // 5. 打开主界面
            FrmMain main = new FrmMain();
            main.Show();
            this.Hide();  // 隐藏登录窗
        }

        private void btn_Register_Click(object sender, EventArgs e)
        {
            // 打开注册窗（ShowDialog 表示以对话框模式弹出）
            FrmRegister reg = new FrmRegister();
            reg.ShowDialog();
        }
    }
}
```

```csharp
using Microsoft.Data.SqlClient;

namespace 转一转校园二手物品交易系统
{
    public partial class FrmRegister : Form
    {
        public FrmRegister()
        {
            InitializeComponent();
        }

        private void btn_Submit_Click(object sender, EventArgs e)
        {
            // ----- 1. 输入校验 -----
            if (string.IsNullOrWhiteSpace(txt_UserName.Text))
            {
                lbl_Tip.Text = "请输入用户名";
                return;
            }
            if (string.IsNullOrWhiteSpace(txt_Pwd.Text))
            {
                lbl_Tip.Text = "请输入密码";
                return;
            }
            if (txt_Pwd.Text != txt_ConfirmPwd.Text)
            {
                lbl_Tip.Text = "两次密码输入不一致";
                return;
            }

            // ----- 2. 检查用户名是否已存在 -----
            string checkSql = "SELECT COUNT(*) FROM users WHERE username=@u";
            SqlParameter[] checkPs = { new SqlParameter("@u", txt_UserName.Text.Trim()) };
            int exists = (int)SQLHelper.Scalar(checkSql, checkPs);
            if (exists > 0)
            {
                lbl_Tip.Text = "该用户名已被注册";
                return;
            }

            // ----- 3. 插入新用户 -----
            string sql = "INSERT INTO users (username, password, phone) VALUES (@u, @p, @ph)";
            SqlParameter[] ps = {
                new SqlParameter("@u", txt_UserName.Text.Trim()),
                new SqlParameter("@p", txt_Pwd.Text),
                new SqlParameter("@ph", string.IsNullOrEmpty(txt_Phone.Text) ? (object)DBNull.Value : txt_Phone.Text)
            };
            SQLHelper.Exec(sql, ps);

            MessageBox.Show("注册成功，请登录！");
            this.Close();  // 关闭注册窗，回到登录界面
        }

        private void btn_Cancel_Click(object sender, EventArgs e)
        {
            this.Close();
        }
    }
}
```

```csharp
using Microsoft.Data.SqlClient;
using System.Data;

namespace 转一转校园二手物品交易系统
{
    public partial class FrmGoodsList : Form
    {
        public FrmGoodsList()
        {
            InitializeComponent();
        }

        private void FrmGoodsList_Load(object sender, EventArgs e)
        {
            LoadData("");
        }

        /// <summary>
        /// 加载商品数据，支持关键字搜索
        /// </summary>
        private void LoadData(string keyword)
        {
            string sql = @"
                SELECT g.goods_id, g.title, g.price, g.status,
                       c.category_name, u.username AS seller_name
                FROM goods g
                JOIN categories c ON g.category_id = c.category_id
                JOIN users u ON g.seller_id = u.user_id
                WHERE g.status = '在售'
                  AND (@k = '' OR g.title LIKE '%' + @k + '%')
                ORDER BY g.created_time DESC";

            SqlParameter[] ps = { new SqlParameter("@k", keyword ?? "") };
            DataTable dt = SQLHelper.Query(sql, ps);
            dgv_Goods.DataSource = dt;
        }

        private void btn_Search_Click(object sender, EventArgs e)
        {
            LoadData(txt_Search.Text.Trim());
        }

        private void btn_Refresh_Click(object sender, EventArgs e)
        {
            txt_Search.Text = "";
            LoadData("");
        }

        private void btn_ViewDetail_Click(object sender, EventArgs e)
        {
            if (dgv_Goods.CurrentRow == null)
            {
                MessageBox.Show("请先选中一个商品");
                return;
            }

            // 从当前选中的行读取 goods_id
            int goodsId = Convert.ToInt32(dgv_Goods.CurrentRow.Cells["ColId"].Value);
            // 打开详情窗体（传商品ID）
            FrmGoodsDetail detail = new FrmGoodsDetail(goodsId);
            detail.ShowDialog();

            // 关闭详情后刷新列表
            LoadData(txt_Search.Text.Trim());
        }

        private void dgv_Goods_CellDoubleClick(object sender, DataGridViewCellEventArgs e)
        {
            // 双击行相当于点击查看详情
            btn_ViewDetail_Click(sender, e);
        }
    }
```

```csharp
using Microsoft.Data.SqlClient;
using System.Data;

namespace 转一转校园二手物品交易系统
{
    public partial class FrmGoodsDetail : Form
    {
        private int _goodsId;    // 存从外面传进来的商品ID
        private int _sellerId;   // 存卖家ID（下单时需要）

        /// <summary>
        /// 构造函数：调用时必须传入商品ID
        /// </summary>
        public FrmGoodsDetail(int goodsId)
        {
            InitializeComponent();
            _goodsId = goodsId;
        }

        private void FrmGoodsDetail_Load(object sender, EventArgs e)
        {
            // 根据商品ID查询完整信息
            string sql = @"
                SELECT g.*, c.category_name, u.username AS seller_name
                FROM goods g
                JOIN categories c ON g.category_id = c.category_id
                JOIN users u ON g.seller_id = u.user_id
                WHERE g.goods_id = @id";

            DataTable dt = SQLHelper.Query(sql, new[] { new SqlParameter("@id", _goodsId) });

            if (dt.Rows.Count == 0)
            {
                MessageBox.Show("商品不存在");
                this.Close();
                return;
            }

            DataRow row = dt.Rows[0];

            // 填充界面
            _sellerId = Convert.ToInt32(row["seller_id"]);
            lbl_Title.Text = row["title"].ToString();
            lbl_Price.Text = "￥" + Convert.ToDecimal(row["price"]).ToString("F2");
            lbl_Seller.Text = "卖家：" + row["seller_name"];
            lbl_Category.Text = "分类：" + row["category_name"];
            lbl_Time.Text = "发布时间：" + Convert.ToDateTime(row["created_time"]).ToString("yyyy-MM-dd");
            rtb_Desc.Text = row["description"]?.ToString() ?? "";

            // 加载图片（如果 URL 不为空）
            string imgUrl = row["image_url"]?.ToString();
            if (!string.IsNullOrEmpty(imgUrl))
            {
                try { pic_Goods.Load(imgUrl); }
                catch { pic_Goods.Image = null; }  // 加载失败就显示空白
            }

            // 按钮控制：不能买自己的商品，已售的不能买
            if (_sellerId == Program.CurrentUserId)
            {
                btn_Order.Enabled = false;
                btn_Order.Text = "不能购买自己的商品";
            }
            else if (row["status"].ToString() != "在售")
            {
                btn_Order.Enabled = false;
                btn_Order.Text = "该商品已售出";
            }
        }

        private void btn_Order_Click(object sender, EventArgs e)
        {
            // 确认对话框
            if (MessageBox.Show("确认购买该商品？", "下单确认", MessageBoxButtons.YesNo) == DialogResult.No)
                return;

            // 1. 插入订单
            string sql1 = "INSERT INTO orders (goods_id, buyer_id, seller_id) VALUES (@g, @b, @s)";
            SqlParameter[] ps1 = {
                new SqlParameter("@g", _goodsId),
                new SqlParameter("@b", Program.CurrentUserId),
                new SqlParameter("@s", _sellerId)
            };
            SQLHelper.Exec(sql1, ps1);

            // 2. 更新商品状态为"已售"
            string sql2 = "UPDATE goods SET status='已售' WHERE goods_id=@id";
            SQLHelper.Exec(sql2, new[] { new SqlParameter("@id", _goodsId) });

            MessageBox.Show("下单成功！请在「我的订单」中查看");
            this.Close();
        }
    }
}
```

```csharp
using Microsoft.Data.SqlClient;
using System.Data;

namespace 转一转校园二手物品交易系统
{
    public partial class FrmPublishGoods : Form
    {
        public FrmPublishGoods()
        {
            InitializeComponent();
        }

        private void FrmPublishGoods_Load(object sender, EventArgs e)
        {
            // 加载分类列表到下拉框
            DataTable dt = SQLHelper.Query("SELECT * FROM categories ORDER BY category_id");
            cbo_Category.DisplayMember = "category_name";   // 界面上显示的名称
            cbo_Category.ValueMember = "category_id";       // 实际存的值
            cbo_Category.DataSource = dt;
        }

        private void btn_Submit_Click(object sender, EventArgs e)
        {
            // 校验
            if (string.IsNullOrWhiteSpace(txt_GoodsTitle.Text))
            {
                MessageBox.Show("请输入商品名称");
                return;
            }
            if (nud_Price.Value <= 0)
            {
                MessageBox.Show("请输入正确的价格");
                return;
            }
            if (cbo_Category.SelectedIndex < 0)
            {
                MessageBox.Show("请选择商品分类");
                return;
            }

            // 插入数据库
            string sql = @"
                INSERT INTO goods (title, price, description, image_url, category_id, seller_id)
                VALUES (@t, @p, @d, @i, @c, @s)";

            SqlParameter[] ps = {
                new SqlParameter("@t", txt_GoodsTitle.Text.Trim()),
                new SqlParameter("@p", nud_Price.Value),
                new SqlParameter("@d", string.IsNullOrEmpty(rtb_Desc.Text) ? (object)DBNull.Value : rtb_Desc.Text),
                new SqlParameter("@i", string.IsNullOrEmpty(txt_ImageUrl.Text) ? (object)DBNull.Value : txt_ImageUrl.Text),
                new SqlParameter("@c", cbo_Category.SelectedValue),
                new SqlParameter("@s", Program.CurrentUserId)
            };

            SQLHelper.Exec(sql, ps);
            MessageBox.Show("商品发布成功！");

            // 清空表单
            txt_GoodsTitle.Clear();
            nud_Price.Value = 0;
            rtb_Desc.Clear();
            txt_ImageUrl.Clear();
        }
    }
}
```

# 第5章 SQLHelper 详解

## 5.1 完整代码（逐行注释）

```csharp
using Microsoft.Data.SqlClient;
using System.Configuration;   // 读 App.config 需要
using System.Data;

namespace 转一转校园二手物品交易系统
{
    /// <summary>
    /// 数据库工具类 —— 所有窗体都通过它操作数据库
    /// 相当于 Java 里的 JdbcUtil
    /// </summary>
    public class SQLHelper
    {
        // 从 App.config 里读取连接字符串
        // ConfigurationManager.ConnectionStrings["SecondHandDB"] 对应 App.config 里的 name
        private static readonly string connStr =
            ConfigurationManager.ConnectionStrings["SecondHandDB"].ConnectionString;

        /// <summary>
        /// 执行 SELECT 查询，返回 DataTable（相当于 Java 的 List<Map>）
        /// </summary>
        /// <param name="sql">SQL 语句（可以带 @参数）</param>
        /// <param name="ps">参数数组（没有参数就传 null）</param>
        /// <returns>查询结果表</returns>
        public static DataTable Query(string sql, SqlParameter[] ps = null)
        {
            // DataTable 相当于 Java 的 List<HashMap>
            DataTable dt = new DataTable();

            // using 语句：代码块结束时自动关闭数据库连接（相当于 Java 的 try-with-resources）
            using (SqlConnection conn = new SqlConnection(connStr))
            {
                conn.Open();  // 打开连接

                using (SqlCommand cmd = new SqlCommand(sql, conn))
                {
                    // 如果传了参数，就加到命令里（防止 SQL 注入）
                    if (ps != null)
                        cmd.Parameters.AddRange(ps);

                    // SqlDataAdapter 相当于 Java 的 ResultSet → 自动把结果填到 DataTable 里
                    new SqlDataAdapter(cmd).Fill(dt);
                }
            }

            return dt;
        }

        /// <summary>
        /// 执行 INSERT / UPDATE / DELETE，返回受影响的行数
        /// </summary>
        /// <returns>影响的行数（大于0表示成功）</returns>
        public static int Exec(string sql, SqlParameter[] ps = null)
        {
            using (SqlConnection conn = new SqlConnection(connStr))
            {
                conn.Open();
                using (SqlCommand cmd = new SqlCommand(sql, conn))
                {
                    if (ps != null)
                        cmd.Parameters.AddRange(ps);

                    // ExecuteNonQuery 执行非查询语句，返回影响行数
                    return cmd.ExecuteNonQuery();
                }
            }
        }

        /// <summary>
        /// 执行查询，返回第一行第一列的值（用于 COUNT、SUM 等）
        /// </summary>
        /// <returns>Object 类型，需要自己转成想要的类型</returns>
        public static object Scalar(string sql, SqlParameter[] ps = null)
        {
            using (SqlConnection conn = new SqlConnection(connStr))
            {
                conn.Open();
                using (SqlCommand cmd = new SqlCommand(sql, conn))
                {
                    if (ps != null)
                        cmd.Parameters.AddRange(ps);

                    // ExecuteScalar 返回第一行第一列
                    return cmd.ExecuteScalar();
                }
            }
        }
    }
}
```

## 5.2 三种方法使用对比

| 方法 | 什么时候用 | 返回值 | Java 类比 |
|------|-----------|--------|-----------|
| `Query` | SELECT 查列表、查详情 | `DataTable`（表） | `List<Map>` |
| `Exec` | INSERT、UPDATE、DELETE | `int`（影响行数） | `int` |
| `Scalar` | COUNT、SUM 等聚合 | `object`（单个值） | `Object` |

### 使用示例

```csharp
// 1. Query 示例：查商品列表
DataTable dt = SQLHelper.Query("SELECT * FROM goods", null);
foreach (DataRow row in dt.Rows)
{
    string title = row["title"].ToString();
}

// 2. Exec 示例：插入新用户（带参数）
string sql = "INSERT INTO users (username, password) VALUES (@u, @p)";
SqlParameter[] ps = {
    new SqlParameter("@u", "张三"),
    new SqlParameter("@p", "123456")
};
int rows = SQLHelper.Exec(sql, ps);  // rows = 1 表示成功

// 3. Scalar 示例：统计用户数
int count = (int)SQLHelper.Scalar("SELECT COUNT(*) FROM users");
```

```java
PreparedStatement ps = conn.prepareStatement(sql);
ps.setString(1, username);
ps.setString(2, password);
ResultSet rs = ps.executeQuery();
```

```csharp
// C# 写法
string sql = "SELECT * FROM users WHERE username=@u AND password=@p";
SqlParameter[] ps = {
    new SqlParameter("@u", username),
    new SqlParameter("@p", password)
};
DataTable dt = SQLHelper.Query(sql, ps);
```

## C.3 三层架构 vs 扁平结构

```java
// Java 三层架构（你的习惯）
Controller → Service → DAO → MyBatis/JDBC

// C# 极速版（扁平结构）
WinForms 窗体 → SQLHelper → SqlClient
```

扁平结构就是：**把 Service 和 DAO 的代码直接写进按钮的 Click 事件里。** 省掉了中间层，开发速度快，但代码不够"优雅"。对 2 周的课程设计来说完全够用。

# 附录D：调试入门

> **调试是写程序最重要的技能。学会调试，你找 Bug 的效率提升 10 倍。**

## D.1 断点（Breakpoint）

断点就是让程序**暂停**在你指定的那一行，方便你看当前变量的值。

**怎么加断点：**
```
在代码行号左边灰色区域点一下 → 出现红色圆点
再点一下取消
```

**怎么跑到断点：**
```
按 F5 运行程序 → 执行到断点那一行时，VS 自动停住
（程序进入"中断模式"，VS 窗口变黄）
```

## D.2 单步执行

程序停在断点后：

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `F5` | 继续运行 | 跑到下一个断点，没有就运行完 |
| `F10` | 逐过程 | 执行当前行，不进入方法内部 |
| `F11` | 逐语句 | 如果当前行调用了方法，进入方法内部 |
| `Shift+F5` | 停止调试 | 结束程序 |

## D.3 看变量值

程序停在断点时：
- **鼠标悬停**在变量名上 → 显示当前值
- **底部"局部变量"窗口** → 显示所有局部变量的值
- **底部"即时窗口"** → 可以输入表达式，回车看结果（如输入 `txt_UserName.Text`）

## D.4 常见调试场景

**场景 1：登录时报错**
```
1. 在 btn_Login_Click 第一行加断点（点左边的灰色区域）
2. 按 F5 运行，输入账号密码，点登录
3. 程序停住 → 鼠标悬停看 txt_UserName.Text 和 txt_Pwd.Text 的值
4. 按 F10 逐行执行，看哪一步报错
```

**场景 2：DataGridView 没数据**
```
1. 在 LoadData 方法的 dt 赋值那一行加断点
2. 运行到断点时，鼠标悬停看 dt
3. 如果 dt.Rows.Count = 0 → SQL 没查到数据
4. 如果 dt 有数据但表格不显示 → DataGridView 列设置有问题
```

**场景 3：找不到控件**
```
在代码中使用 txt_UserName 时报红，但明明拖了控件 → 检查 .Designer.cs
在最底部找有没有 private TextBox txt_UserName; 这一行
```
